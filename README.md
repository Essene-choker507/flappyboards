## 🔧 Tips for a better result

- Keep messages short
- Use all caps if the display style looks better that way
- Avoid too many symbols
- Test one line at a time
- Use strong contrast between text and background

If the board flips too fast, slow the animation down. If it feels hard to read, make the letters larger.

## 🛠️ Troubleshooting

### The file does not open

- Check that the download finished
- Try running it again
- If it is inside a ZIP file, extract it first
- Right-click the file and choose Run as administrator

### Windows blocks the file

- Open the file properties
- Look for an Unblock option
- Apply it if present
- Run the file again

### The TV shows a blank screen

- Check the HDMI cable
- Make sure the TV input is correct
- Move the app window onto the TV
- Switch the app to full screen

### The text looks cut off

- Make the font smaller
- Shorten the message
- Change the screen resolution
- Test with a single word

### The app is hard to find after download

- Open your Downloads folder
- Sort by date
- Look for the newest file
- Use the same folder each time
